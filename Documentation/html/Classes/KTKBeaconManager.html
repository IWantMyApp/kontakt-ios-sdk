KTKBeaconManager Class Reference

Inherits from NSObject
Declared in KTKBeaconManager.h

Overview

Entry point to monitoring and ranging devices.

Interface Example:

#import <KontaktSDK/KontaktSDK.h>

@interface SomeInterface () < KTKBeaconManagerDelegate >

@property (strong) KTKBeaconManager *manager

@end

Usage:

// Core Location Beacon Region with Kontakt.io Proximity UUID
KTKBeaconRegion *beaconRegion = [KTKBeaconRegion alloc] initWithProximityUUID:@"f7826da6-4fa2-4e98-8024-bc5b71e0893e"
                                                                 identifier:@"identifier"];

// You can also use KTKBeaconRegion Category/Extension methods
beaconRegion = [KTKBeaconRegion ktk_standardProximityRegion];


self.manager = [[KTKBeaconManager alloc] initWithDelegate: self];

// When using iBeacons this should be called
[self.manager requestLocationAlwaysAuthorization];

// Start ranging devices or monitoring or both
[self.manager startRangingDevicesInRegion: beaconRegion];
[self.manager startMonitoringForRegion: beaconRegion];
